ShobdoHotao — home background images
====================================

Put your exported background design(s) in THIS folder, named exactly:

    background_1.png      (shown as the home background)
    background_2.png      (optional — if present, crossfades with _1)
    background_3.png      (optional — and so on)

Accepted extensions: .png  .jpg  .jpeg  .webp   (png recommended)
Recommended size:     16:9, at least 1920x1080 (2560x1440 for big screens)

How they are used:
- Drawn CRISP (no blur), full-bleed, with only a light veil so the white
  title/text stays readable.
- More than one image -> they crossfade slowly.
- No image here -> the home background is a plain deep-blue fill.

These are your own artwork, so there is no third-party licensing concern.
